App Info

An item which describes an application. Typically this would be one item in a list of applications. The heading is the application name. The info below the heading typically provides subtext of less importance, e.g. date of submission. The call-to-action link is typically to process a pending application, e.g. Review.

Approved

An application which has been approved and has a status indicating whether or not it has been verified via email.
