Buttons

Buttons are used for calls to actions to start a journey or to submit information on a form. Simply place the .button selector on your <button> element. If you are using an <a> element please also apply the role="button" attribute.

<button class="button [modifier class]">Button text</button>

.button--get-started

For a button to start a user journey 
<button class="button button--get-started">Button text</button>

.button--secondary

A secondary button 
<button class="button button--secondary">Button text</button>

.button--alert

An alert button 
<button class="button button--alert">Button text</button>

.button--small

A smaller button 
<button class="button button--small">Button text</button>

.button--table

A button within a table 
<button class="button button--table">Button text</button>

.button.button--link

To make a button appear like a link 
<button class="button button button--link">Button text</button>
