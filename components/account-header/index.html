This is a work in progress – view more information on GitHub.

When to use this component

If your service is part of the personal tax account.

When not to use this component

If your service is not part of the personal tax account. Use the header instead.

How it works

The account header contains:

  • the HMRC cookie banner
  • GOV.UK header
  • the Account menu
  • the language selector if your service is available in Welsh
